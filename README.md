# 📱 MeteoNode — Android-приложение

Мобильное приложение для управления и мониторинга метеостанции MeteoNode. Позволяет следить за показателями датчиков, настраивать пороги, получать уведомления и просматривать статистику.

---

## ✨ Возможности

| Раздел | Описание |
|--------|----------|
| 🏠 **Главный экран** | Текущие показатели: температура, влажность, давление, CO₂, TVOC, AQI |
| 📈 **Статистика** | Графики за 24ч/неделю/месяц/год с анимацией |
| ⚙️ **Настройки** | Пороги, яркость, недельное расписание |
| 🔔 **Уведомления** | Push-уведомления при превышении порогов |
| 🔍 **Автопоиск** | Находит устройство в локальной сети по IP и mDNS |
| 🎨 **Темы** | Светлая, тёмная, системная |
| 📡 **Фоновый сервис** | Собирает статистику даже когда приложение свёрнуто |

---

## 🛠 Технологии

| Компонент | Библиотека |
|-----------|------------|
| Язык | Kotlin |
| Сеть | OkHttp 4 |
| Графики | MPAndroidChart |
| Навигация | Navigation Component |
| UI | Material Design 3 + ViewBinding |
| Асинхронность | Kotlin Coroutines |
| Фоновый мониторинг | Foreground Service |
| Хранение | SharedPreferences |
